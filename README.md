# EverController

Minecraft sunucuları için gelişmiş kontrol ve loglama eklentisi.

## Özellikler
- Drop / Pickup loglama
- Container break takibi
- Discord webhook entegrasyonu

## Kurulum
1. Plugin jar dosyasını `plugins/` klasörüne at
2. Sunucuyu başlat
3. `config.yml` düzenle
4. `/plugman reload EverController` veya `/stop`

## Gereksinimler
- Java 17+
- Paper / Spigot
